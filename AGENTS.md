# TeaLixir Website Agents

## Project Description

This project is a website for the TeaLixir iOS app. The website is built using the 
Hugo static site generator, and hosted on the user's preferred hosting platform.

### iOS App Information

The TeaLixir iOS app is a tea tracking app. The app allows users to track their teas, 
and provides recommendations for tea based on their preferences. The app is available
in private beta, and coming soon to the App Store.

Features include:
- Track specific teas
  - Tea type (black, green, herbal, etc.)
  - Caffiene level
  - Tasting notes
  - Origin/Estate/harvest season/organic/fair trade (optional)
  - Brand and seller (optional)
  - Rating
  - Image
  - Brewing instructions
- Simplified data entry
  - Take pictures of the tea bag to enter data
- Tea Preference Analysis
  - The app learns from your tea likes and dislikes and provides recommendations
  - View a profile of your tea preferences
- Stores data to Apple Cloud so you can access it from multiple devices or have the data ready for you when you switch to a new phone
